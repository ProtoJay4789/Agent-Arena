# AEG — Agent Economy Gaming

A DeFi trading simulation game that teaches real crypto mechanics without real money.

## Concept

Players start with spot trading capital and unlock leverage tiers through demonstrated skill. Bot advisors guide the journey — listen to them and progress steadily, ignore them and face liquidation.

## Core Features

- **Spot Trading Sim** — start with base capital, build through smart trades
- **Leverage Tiers** — unlock 2x → 5x → 10x → 20x through reputation
- **Bot Advisors** — contextual guidance that reacts to player decisions
- **Gas Penalty System** — bad trades cost money, unpaid fees bleed reputation
- **Liquidation Mechanics** — realistic thresholds that teach without real loss

## Why It Works

> "Parking ticket, not a boot." — You still play, you just owe money.

The gas penalty system creates natural urgency. Rep decay on unpaid fees means players want to clear their debt. No rage quits — just mounting pressure to play smarter.
